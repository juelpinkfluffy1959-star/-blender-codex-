# Blender Codex Materials

This repository is a focused copy of the Blender-related materials from:

https://github.com/jianbaorui07-dot/Codex-Integration-with-Creative-Industry-Software

Copied source branch:

`codex/blender-drawing-learning-data`

## Contents

- `examples/blender_bridge/`: Blender bridge manifest, status metadata, and read-only environment probe.
- `docs/04-codex-blender.md`: Blender bridge integration notes.
- `docs/blender-drawing-learning/`: redacted Blender learning notes, scene profiles, case studies, and plugin/tool specs.
- `scripts/analyze_blender_learning_assets.py`: build a redacted learning index from a local Blender asset library.
- `scripts/blender_scene_profile.py`: run inside Blender to generate redacted scene profiles.
- `scripts/generate_blender_case_notes.py`: generate per-case notes from scene profiles.

No private `.blend` files, textures, HDR files, render cache, or local absolute paths are included.
