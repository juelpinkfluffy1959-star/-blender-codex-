# 中文用途索引

这个索引用中文说明仓库里每个主要入口的用途。当前仓库只发布和 **Codex 接入各类本地创意软件** 有关的公开安全内容，覆盖 ComfyUI、CAD / AutoCAD、Photoshop、Illustrator、Blender、剪映 / CapCut。这里的“接入”现在分两层：StarBridge 探针负责确认本机环境和安全边界，MCP stdio tools 负责把已收敛的安全能力交给 AI 客户端调用；这不等于所有软件都已经完整自动化。

## 状态和命令

| 原文件 | 中文用途 |
| --- | --- |
| `scripts/collect_bridge_status.py` | 读取 `examples/*_bridge/bridge.json`，输出 Markdown 或 JSON manifest |
| `examples/bridge_status.py` | 探测 ComfyUI / Blender / CAD / Photoshop / Illustrator / 剪映 / CapCut 本机环境 |
| `starbridge_mcp/mcp_server.py` | 真正 MCP stdio server，向 AI 客户端暴露 `tools/list` 和 `tools/call` |
| `starbridge_mcp/core/tool_registry.py` | StarBridge 能力注册表，标注每个工具的成熟度、风险和确认要求 |
| `scripts/security_check.py` | 检查禁止提交的资产类型、敏感路径和密钥样式文本 |
| `tests/` | 离线测试：manifest schema、脚本路径、安全检查、ComfyUI workflow 校验、CAD 解析、报告格式 |

常用命令：

```powershell
npm.cmd run status:manifest
npm.cmd run status:manifest:json
npm.cmd run status:probe:json
npm.cmd run starbridge:mcp
npm.cmd run starbridge:tools:safe
npm.cmd run comfy:probe
npm.cmd run comfy:txt2img -- --prompt "a quiet futuristic tea house" --ckpt "<checkpoint-name>"
npm.cmd run photoshop:diagnose
npm.cmd run security:check
npm.cmd test
```

## 不发布内容

- 历史网页 demo、虚拟宠物 demo、PPT 工作区和无关临时输出。
- 报告生成临时产物、图片素材、样式参考文档。
- 输出目录、缓存目录、日志和临时文件。
- 模型文件、生成图片、私有 `.blend`、客户 DWG、PSD/AI 工程、剪映草稿、导出视频、Photoshop/Illustrator 安装路径和本机素材路径。
- 账号、密码、验证码、Cookie、Token、OAuth 缓存、支付信息和浏览器资料。
