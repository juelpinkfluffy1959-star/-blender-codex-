# Blender 绘制图学习笔记

这批素材最适合学习三件事：角色/道具如何建起来，材质和贴图如何让模型“像作品”，灯光与相机如何把模型变成一张图。

## 观察到的绘制逻辑

### 1. 从大形开始，而不是从细节开始

多个角色、车辆和道具案例都大量使用 `Mirror`、`Subsurf`、`Solidify`。这说明作者不是一开始就雕细节，而是先做轮廓、比例和对称结构。

学习方式：

- 先把模型切到白模或材质预览，观察剪影。
- 找主要体块：头、躯干、四肢、车身、机翼、耳机外壳、咖啡机主体。
- 看 `Mirror` 是否存在，判断作者是不是只建了一半再镜像。
- 看 `Subsurf` 是否存在，理解低模控制点如何变成圆滑表面。
- 看 `Solidify` 是否存在，学习衣服、壳体、叶片、护甲的厚度处理。

### 2. 角色绘制依赖“身体 + 衣物 + 骨骼 + 展示姿势”

角色文件里有大量 `ARMATURE` 和 `SUBSURF`，二次元/次时代角色通常不是单个网格，而是身体、头发、衣物、装饰、武器、表情、骨骼的组合。

学习方式：

- 先只看身体比例，不看材质。
- 再看衣物厚度和边缘处理，重点观察 `Solidify`。
- 再看头发、装饰、武器如何拆成独立对象。
- 最后看骨骼姿势和相机角度，理解角色为什么在这个角度好看。

可练习题：

- 复制一个角色的站姿构图，用基础人体块面重建。
- 只临摹衣物厚度和边缘，不临摹整个人。
- 用三盏灯重新打一个角色展示图：主光、轮廓光、环境光。

### 3. 硬表面绘制靠边缘高光

车辆、直升机、机器人、耳机、咖啡机这类案例里，`Bevel`、`Mirror`、`Subsurf`、`Weighted Normal` 一类结构很重要。硬表面不是把边做得无限锐，而是用适当倒角让光能挂在边缘上。

学习方式：

- 看倒角宽度和物体尺度是否匹配。
- 看高光是否沿边缘形成连续线。
- 关掉材质后看白模，判断形体是否仍然清楚。
- 用 HDR 或大面积 Area Light 检查反射。

可练习题：

- 做一个简单耳机外壳，只练镜像、倒角和材质反射。
- 做一个车身白模，先不贴图，只用倒角和灯光让轮廓成立。
- 给一个硬表面模型换三种材质：哑光塑料、亮面金属、喷漆车漆。

### 4. 材质学习要拆贴图角色

素材里有大量 JPG/PNG/TIF 贴图和节点材质。学习时不要只问“这张贴图好不好看”，而要问它在材质系统里扮演什么角色。

常见贴图角色：

- Base Color：决定固有色和图案。
- Roughness：决定表面粗糙度和反射散不散。
- Normal/Bump：决定小凹凸，不改变真实轮廓但改变受光。
- Metallic：决定金属/非金属。
- Alpha/Opacity：用于树叶、头发片、镂空和半透明。
- Emission：用于发光纹理、电子屏、魔法效果。

学习方式：

- 在材质节点里逐个断开贴图，看画面少了什么。
- 把 roughness 拉到极端，观察材质从干到湿、从哑到亮的变化。
- 把 normal/bump 关掉，观察细节是否还成立。

### 5. 灯光和 HDR 是“绘制图”的气氛层

108 个 HDR/EXR 环境文件说明这批素材不是只关心模型，还重视不同环境光对材质和画面的影响。HDR 环境特别适合车辆、金属、玻璃、产品、角色展示。

学习方式：

- 同一个模型换 3 个 HDR，看反射和阴影如何变化。
- 用世界环境做基础光，再加一盏主光控制视觉中心。
- 对角色用冷暖对比，对产品用大面积柔光，对机械用清晰边缘光。

### 6. 相机不是记录工具，是构图工具

25 个 `.blend` 中抽到 31 个相机。一个文件有多个相机，往往代表作者在试不同构图：全身、半身、特写、产品视角、英雄视角、展示视角。

学习方式：

- 记录每个相机的焦段和位置。
- 看模型是否为某个镜头专门摆姿势。
- 学会用 70mm 以上压缩空间做角色/产品展示，用 24-35mm 做动态和空间感。

## 推荐学习顺序

1. `练习素材`：保龄球、咖啡机、直升机、女法师。先临摹小案例，学习物体组织。
2. `模型`：机器人、越野车、人头、全身模型带材质。学习白模、硬表面、人体和材质。
3. `5-Hdr环境`：把同一模型放进不同 HDR，专门练灯光气氛。
4. `6-人物模型`：拆角色结构，学习比例、衣物、绑定、材质和姿势。
5. `8-游戏人物`：作为风格和角色库参考，先分类再选择性解包学习。

## 我的结论

Blender 绘制图不是单一动作，而是一条管线：

```text
参考/构思 -> 大形建模 -> 对称和细分 -> 厚度/倒角 -> UV/贴图 -> 材质节点 -> 灯光/HDR -> 相机 -> 渲染 -> 后期微调
```

这批数据最有价值的地方，是它同时覆盖了角色、硬表面、材质、HDR 和练习场景。正确学法不是一次打开所有文件，而是每次只研究一个问题：今天学镜像建模，明天学倒角高光，后天学角色打光。这样才能把素材库变成自己的 Blender 绘制能力。
