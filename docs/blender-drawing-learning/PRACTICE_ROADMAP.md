# Blender 绘制图练习路线

这条路线按“看懂别人怎么做 -> 复建一个小功能 -> 完成一张图”的方式组织。

## 阶段 1：看懂文件

目标：打开一个 `.blend` 后知道应该观察什么。

练习：

- 统计场景里有多少 Mesh、灯光、相机、材质。
- 找出主模型、辅助模型、背景和装饰物。
- 关闭材质，只看白模轮廓。
- 关闭灯光，只看模型本身是否成立。
- 切换相机，判断每个相机服务于什么画面。

产出：

- 一张场景结构截图。
- 一段 100 字说明：这个文件主要靠什么成立，是造型、材质、灯光还是构图。

## 阶段 2：临摹大形

目标：不追求细节，先学会把形体搭出来。

练习：

- 用立方体、球体、圆柱重建咖啡机/耳机/直升机/车辆的大体块。
- 用 `Mirror` 做对称物体。
- 用 `Subsurf` 让低模变顺。
- 用 `Solidify` 给薄片、衣物、壳体增加厚度。

产出：

- 一个白模版本。
- 一个带基础灰材质的版本。

## 阶段 3：学习硬表面高光

目标：让机械、车辆、产品边缘在灯光下好看。

练习：

- 给模型关键边加 `Bevel`。
- 对比无倒角、小倒角、大倒角的高光差异。
- 给模型加金属、塑料、车漆三种材质。
- 用一张 HDR 和一盏 Area Light 测试反射。

产出：

- 三张同角度材质对比图。
- 一份倒角参数记录。

## 阶段 4：学习角色结构

目标：知道角色不是一个网格，而是一组可分层研究的对象。

练习：

- 分层观察身体、衣服、头发、装饰、武器。
- 查看 `Armature` 如何影响姿势。
- 找衣服和装甲上的 `Solidify`。
- 找身体和头发上的 `Subsurf` 或平滑处理。

产出：

- 一张角色分层表。
- 一个简化角色白模。

## 阶段 5：学习材质节点

目标：理解贴图和节点如何共同决定画面。

练习：

- 找出 Base Color、Roughness、Normal、Metallic、Emission。
- 逐个断开贴图，观察画面变化。
- 替换一张 roughness 或 normal，观察材质气质变化。
- 给同一个白模做三套材质。

产出：

- 一张材质节点截图。
- 三张材质风格图。

## 阶段 6：学习灯光和相机

目标：把模型变成一张完成度更高的图。

练习：

- 用 HDR 做环境光。
- 加主光、轮廓光、补光。
- 切换 35mm、50mm、85mm 焦段对比构图。
- 分别做全景、半身、特写、产品英雄角度。

产出：

- 四张构图图。
- 一份灯光说明：主光方向、颜色、强度、环境光来源。

## 阶段 7：复建完整小作品

目标：独立完成一张 Blender 绘制图。

推荐题目：

- 科幻耳机产品图。
- 蓝紫灯光机器人图。
- 女法师角色展示图。
- 越野车硬表面展示图。
- 程序化西瓜材质图。

最终产出：

- 一个 `.blend` 练习文件。
- 一张最终渲染图。
- 一份过程记录：建模、材质、灯光、相机、渲染设置。

## 给后续数据投喂的格式

每次新增素材或案例时，建议补充这些信息：

| 字段 | 说明 |
| --- | --- |
| 案例名 | 文件或素材包名称 |
| 类型 | 角色、产品、车辆、环境、材质、灯光 |
| 想学什么 | 建模、材质、灯光、相机、动画、渲染 |
| 当前问题 | 看不懂节点、不会复建、不会打光、不会出图 |
| 期望产出 | 学习笔记、步骤拆解、复建脚本、渲染方案 |

这样我就能继续把新数据吸收成结构化的 Blender 绘制知识。
